# 版本记录（edustation）

> 本文件记录 **edustation 项目**（即 edustation 整个工程：后端 server + 管理后台 admin + 客户端 client + 共享 shared）的版本号与每次发布的变更说明。
>
> 命名约定：
> - 主版本号（MAJOR）：破坏性架构变更时才升
> - 次版本号（MINOR）：新增模块 / 较大功能集合
> - 修订号（PATCH）：bugfix / 文档
>
> 提交约定：每次发版在 git 上打一个 tag（如 `v0.3.0`），并把本文件同步更新。
