# EduStation 隐私政策

**版本号:1.0.0**
**生效日期:2026 年 06 月 17 日**

EduStation 平台运营方(以下称"我们")深知个人信息对您的重要性,并将依据《中华人民共和国个人信息保护法》《中华人民共和国网络安全法》《信息安全技术 个人信息安全规范》等相关法律法规及监管要求,采取相应的技术和管理措施保护您的个人信息安全。

本《隐私政策》将向您说明:我们会如何收集、使用、存储、共享、转让、公开披露您的个人信息,以及您所享有的权利。**请您在使用我们的产品(或服务)前仔细阅读并充分理解本政策**,在确认理解并同意后再开始使用。

## 一、我们收集的个人信息

### 1.1 您注册账号时收集

- **手机号**(必要):用于账号唯一标识与登录认证
- **姓名 / 昵称**(必要):用于联系与显示
- **密码**(必要):加盐加密存储,我们不掌握明文
- **头像**(可选):用于个人信息展示

### 1.2 您使用服务时收集

| 场景 | 收集字段 | 用途 |
|------|---------|------|
| 学员管理 | 学员姓名 / 性别 / 出生日期 / 监护人关系 | 提供教学与考勤服务 |
| 排课考勤 | 排课时间 / 考勤状态 / 课时消耗 | 课表、消课计算 |
| 订单交易 | 订单号 / 课程产品 / 金额 / 支付方式 / 支付凭证 | 交易记录与对账 |
| 作品上传 | 图片 / 视频 / 文字描述 | 作品展示与家长分享 |
| 互动分享 | 分享渠道 / 时间 / 积分变动 | 积分发放与统计 |
| 招生客户 | 潜客手机号 / 姓名 / 渠道来源 / 跟进记录 | 销售跟单 |
| 设备信息 | IP 地址 / 设备型号 / 操作系统 / 浏览器 | 安全风控与体验优化 |
| 日志信息 | 访问时间 / 访问路径 / 操作记录 | 故障排查与服务改进 |

### 1.3 您主动提供时收集

- **身份证号**(可选):仅在涉及合规审核或机构办学许可时使用
- **位置信息**(可选):仅在使用"导航到机构"等位置相关功能时使用,您可随时关闭
- **通讯录 / 相册**(可选):仅在上传头像 / 作品时按需调用,我们不读取无关内容

### 1.4 我们不会收集的信息

- 您的银行卡号 / 信用卡 CVV 等敏感金融信息(支付走第三方支付平台,我们不接触)
- 您的人脸 / 指纹等生物特征(除非您主动开通刷脸考勤功能,且会单独取得授权)
- 您与他人的私人通讯内容

## 二、我们如何使用您的个人信息

2.1 提供您所需要的服务及客户支持

2.2 满足您的个性化需求(例如根据您所选机构推送相关课程)

2.3 进行内部安全管理和身份验证、安全防范、诈骗监测、存档备份、客户安全服务等

2.4 改进我们的产品和服务,例如根据您使用我们服务时所发生的问题进行错误排查

2.5 遵守法律法规、监管要求,以及响应政府或司法机关的合法要求

2.6 经您额外同意的其他用途

**我们不会将您的个人信息用于未向您告知的其他用途。**如需将信息用于本政策未告知的其他用途,我们会再次征求您的同意。

## 三、我们如何共享、转让、公开披露您的个人信息

### 3.1 共享原则

我们不会与本平台运营方以外的任何公司、组织和个人共享您的个人信息,**除非**:
- 事先获得您的明确同意
- 法律法规规定或司法机关、行政机关合法要求
- 为保护您或社会公众的人身、财产安全免遭侵害
- 您主动公开的信息
- 与我们的关联公司、授权合作伙伴共享(共享方将受到本政策同等约束)

### 3.2 必要的第三方共享

| 第三方 | 共享信息 | 目的 |
|--------|---------|------|
| 第三方支付平台(微信 / 支付宝) | 订单金额 / 订单号 | 收付款处理 |
| 短信服务商 | 您的手机号 | 发送验证码 / 业务通知 |
| 云存储服务商 | 您上传的文件(已加密) | 文件存储 |
| 数据分析服务商 | 脱敏后的访问日志 | 改进产品体验 |

### 3.3 业务转让

随着我们业务的持续发展,我们有可能进行合并、收购、资产转让等交易,我们将告知您相关情形,按照法律法规及不低于本政策所要求的标准继续保护或要求新的持有者继续保护您的个人信息。

### 3.4 公开披露

我们仅会在以下情况下公开披露您的个人信息:
- 获得您明确同意后
- 基于法律的披露:在法律、法律程序、诉讼或政府主管部门强制要求的情况下

## 四、我们如何存储您的个人信息

4.1 **存储地点**:您的个人信息存储于中华人民共和国境内的服务器。如未来涉及向境外提供您的个人信息,我们将单独取得您的同意。

4.2 **存储期限**:我们仅在为您提供本服务之目的所必需的期间内保留您的个人信息,法律法规另有规定的除外。超出必要存储期限后,我们将删除或匿名化处理。

4.3 **存储安全**:
- 使用业界标准的安全技术保护您的个人信息(传输加密 TLS / 存储加密 AES)
- 密码、身份证等敏感字段使用强加密算法(argon2 / AES-256)
- 严格的权限隔离(基于职位的最小权限原则,即"知情即用")
- 定期审计访问日志,异常告警

## 五、您的权利

依据中国相关法律法规与标准,我们保障您对自己的个人信息行使以下权利:

### 5.1 访问权
您有权访问您的个人信息。您可在"我的"页面随时查看您的账号信息、订单记录、课包信息等。

### 5.2 更正权
您有权更正错误或不完整的个人信息。您可在"个人资料"页面自行修改,部分敏感字段需联系机构管理员协助修改。

### 5.3 删除权
您在以下情形下可以向我们提出删除您个人信息的请求:
- 我们处理个人信息的行为违反法律法规
- 我们收集、使用您的个人信息却未征得您的同意
- 我们处理个人信息的行为违反了与您的约定
- 您注销账号
- 我们终止服务

请通过平台客服或机构管理员发起删除请求,我们将在 15 个工作日内响应。

### 5.4 撤回同意权
您有权撤回您此前对我们处理个人信息的授权同意。请注意,撤回同意将影响您正常使用相关服务,但**不影响您撤回前我们基于您的授权所开展的个人信息处理活动的效力**。

### 5.5 注销权
您可通过平台客服注销账号。账号注销后,您的非保留个人信息将被删除或匿名化处理。

### 5.6 投诉权
如您认为我们对您个人信息的处理行为损害了您的合法权益,您可向网信、工信、公安等监管部门投诉,也可向人民法院起诉。

## 六、对未成年人的特别说明

6.1 本平台主要面向校外培训机构、家长、教师群体。原则上,平台不直接面向不满 14 周岁的未成年人。

6.2 学员档案中可能包含不满 14 周岁未成年人的姓名、年龄等信息,这些信息由其监护人(家长)账号代为录入与维护。具体规则见《EduStation 未成年人个人信息处理规则》。
