# EduStation 未成年人个人信息处理规则

**版本号:1.0.0**
**生效日期:2026 年 06 月 17 日**

EduStation 平台运营方(以下称"我们")依据《中华人民共和国未成年人保护法》《中华人民共和国个人信息保护法》《儿童个人信息网络保护规定》等法律法规,制定本《未成年人个人信息处理规则》。

本规则适用于本平台收集、使用、存储、共享、转让、公开披露不满 14 周岁未成年人(以下称"儿童")个人信息的全部活动。

## 一、适用范围与基本原则

1.1 本规则适用于本平台所有涉及儿童个人信息处理的场景,包括但不限于:
- 学员档案(姓名、性别、出生日期等)
- 课程报名、考勤记录
- 学员作品(图片 / 视频 / 文字)
- 积分账户、虚拟宠物等互动数据

1.2 本平台不开设儿童直接登录账号。儿童的所有数据均由其**监护人**(以下称"家长")通过家长账号代为录入、管理、维护。

1.3 我们处理儿童个人信息,遵循以下原则:
- **正当必要原则**:仅收集为提供服务所必需的最少信息
- **明确同意原则**:必须取得家长明示同意
- **知情同意原则**:家长有权随时了解、查询、修改、删除儿童个人信息
- **最小授权原则**:仅授权特定岗位人员(如机构教师、教务)在职责范围内查询
- **安全保护原则**:采用业界最高级别的安全防护措施

## 二、家长授权与责任

2.1 您作为儿童的监护人,在为儿童创建学员档案、办理报名、上传作品时,即视为**您已代为同意**本平台收集、使用儿童的相关个人信息。

2.2 您应当:
- 真实、准确地填写儿童信息
- 关注本规则及隐私政策的更新
- 监督儿童正确使用平台服务(如有)
- 在发现儿童个人信息被泄露、篡改、丢失等情形时,立即联系机构管理员或平台客服

2.3 您撤回授权将影响相关服务的正常使用(例如撤回后,该儿童将无法继续在本机构上课、查看作品等)。

## 三、我们收集的儿童个人信息

### 3.1 必要信息(无此信息无法提供服务)

| 字段 | 用途 |
|------|------|
| 姓名 | 课程报名、考勤、作品归属 |
| 性别 | 教学分班参考 |
| 出生日期 | 课程匹配、班级分级 |
| 监护人(家长账号) | 关联监护关系 |

### 3.2 可选信息(可拒绝提供,不影响基础服务)

| 字段 | 用途 |
|------|------|
| 头像 | 学员档案展示 |
| 所属学校 / 年级 | 班级分组参考 |
| 兴趣爱好 | 个性化课程推荐 |
| 作品(图片 / 视频) | 学习成果展示与分享 |

### 3.3 我们绝不收集的信息

- **生物特征**(人脸、指纹、虹膜、声纹等):除非家长**单独**同意开通"刷脸考勤"等特定功能,且此类信息仅在设备本地处理,不上传服务器
- **身份证号 / 户口本**:平台不要求儿童提供此类信息
- **金融信息**:儿童不参与支付,所有支付均由家长账号完成

## 四、儿童个人信息的使用规则

4.1 我们仅在以下范围内使用儿童个人信息:
- 提供教学相关服务(报名、排课、考勤、作品记录等)
- 与家长沟通教学进度、孩子表现
- 内部统计分析(以匿名化、聚合形式)
- 法律法规要求

4.2 **我们绝不会**:
- 将儿童个人信息用于商业推广 / 精准营销
- 将儿童个人信息卖给任何第三方
- 在公开场合披露儿童身份(作品分享需家长另行授权)

## 五、机构端访问与权限控制

5.1 机构内部哪些角色可以访问儿童信息,通过严格的角色权限控制:

| 角色 | 可见信息范围 |
|------|------------|
| 机构管理员 | 全部学员档案(本机构) |
| 教务 | 全部学员档案(本机构) |
| 老师 | 仅本人任课开班的学员 |
| 财务 | 仅订单与课包相关字段 |

5.2 任何越权访问都会留下审计日志,平台定期审查异常行为。

5.3 机构员工离职后,其访问权限将被立即撤销。

## 六、作品分享的额外授权

6.1 您的孩子在课上完成的作品(画作、视频、表演等)可能被机构上传至本平台。

6.2 默认情况下,作品仅在以下范围内可见:
- 您本人(家长账号)
- 您孩子的任课老师与机构教务
- 机构管理员

6.3 如机构希望将作品用于:
- 公开宣传(机构主页、招生海报、朋友圈营销)
- 比赛、展览投稿
- 媒体报道

则**必须取得您的单独书面授权**(机构应通过线下书面或线上明确勾选方式获得)。

## 七、儿童个人信息的存储与安全

7.1 **存储地点**:中华人民共和国境内服务器。

7.2 **存储期限**:儿童在本机构期间持续存储;离校后保留 3 年(供日后查询作品、咨询),3 年期满后删除或匿名化(法律另有要求的除外)。

7.3 **安全措施**:
- 所有儿童相关数据存储字段在数据库层加密
- 备份数据同等保护
- 访问需通过身份认证 + 权限校验双层关卡
- 定期安全演练与渗透测试

7.4 **泄露应急**:如发生数据泄露,我们将在 72 小时内通知家长与监管部门,并采取补救措施。

## 八、家长的权利

作为儿童监护人,您随时可以:

8.1 **查阅**:登录家长账号查看您孩子的所有信息

8.2 **更正**:通过家长账号修改可改字段;不可改字段联系机构管理员协助

8.3 **删除**:
- 删除单条作品 → 家长账号直接操作
- 删除学员档案 → 联系机构管理员(机构需先结清未消费课包)
- 注销账号 → 联系平台客服

8.4 **撤回授权**:您可随时撤回任何此前授予的非必要授权(如作品公开授权);撤回不影响撤回前已完成的处理

8.5 **投诉与举报**:如发现儿童信息被违规处理,请立即:
- 联系机构管理员;或
- 联系平台客服;或
- 向网信办、教育主管部门举报

## 九、对机构(数据处理者)的要求

9.1 机构入驻本平台时,必须签署《数据处理协议》,承诺:
- 严格按本规则收集、使用儿童信息
- 对接触儿童信息的员工进行隐私培训
- 不得擅自向任何第三方共享儿童信息
- 离职员工权限立即撤销

9.2 平台有权对机构的合规情况进行抽查,违规机构将被警告、限期整改,严重违规者将被强制下线。

## 十、本规则的修订

10.1 我们可能根据法律法规更新或业务调整需要修订本规则。

10.2 重大变更将以显著方式通知家长,并重新征求您的同意。

10.3 本次版本号 1.0.0,生效日期 2026 年 06 月 17 日。

## 十一、联系我们

如您对本规则有任何疑问,或希望行使家长权利,请通过以下方式联系:

- **机构联系**:在「我的 → 机构主页」找到机构联系方式
- **平台客服**:在「我的 → 联系我们」获得平台客服信息
- **监管部门**:网信办举报中心 12377
